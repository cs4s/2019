VR Activities

This page has all of the workshop's activities, split into four main topics: Creating and Sharing VR Scenes, 3D Positioning and Shapes, Images and 3D Models and Interactions and Animations.

Creating and Sharing VR Scenes

In these activities you will learn how to create and view your own websites using Glitch and how to create VR scenes on Glitch with A-Frame VR.

Activities

  • Using Glitch
  • Creating VR with A-Frame

Links

  • Glitch Homepage
  • A-Frame
  • A-Frame School

3D Positioning and Shapes

In these activities you will learn how to use the A-Frame Inspector tool to view and change entities in your VR scenes and how to position 3D shapes when creating VR scenes in A-Frame.

Activities

  • Using the A-Frame Inspector
  • Positioning Shapes in A-Frame

Links

  • A-Frame Inspector Documentation
  • A-Frame Geometry Component Documentation
  • A-Frame Text Component Documentation

Images and 3D Models

In these activities you will learn how to use A-Frame to create a 3D photo viewer and how to download and add 3D models to your VR scenes.
